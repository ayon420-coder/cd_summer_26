# Compiler Design Lab

## Overview

This repository contains my Compiler Design laboratory assignments and practice implementations developed in C++. The purpose of this repository is to explore fundamental concepts of compiler construction, including lexical analysis, finite automata, token recognition, parsing techniques, and syntax analysis.

## Topics Covered

* Lexical Analysis
* Token Recognition
* Comment Detection
* Keyword, Identifier, and Operator Identification
* Finite Automata (DFA/NFA)
* Regular Expressions
* Symbol Table Implementation
* Syntax Analysis
* Parser Construction
* Compiler Design Lab Assignments

## Technologies Used

* C++
* Code::Blocks / GCC Compiler
* Standard Template Library (STL)

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/compiler-design-lab.git
   ```

2. Navigate to the desired assignment folder.

3. Compile the source file:

   ```bash
   g++ filename.cpp -o output
   ```

4. Run the executable:

   ```bash
   ./output
   ```

## Learning Objectives

* Understand the phases of a compiler.
* Implement lexical analyzers and tokenizers.
* Apply finite automata concepts to language processing.
* Learn the basics of parsing and syntax analysis.
* Develop practical compiler construction skills.
